Assignment Test (Submit with in 3days )

Micro Frontend Application  – React.js/Next.js

Objective
Build a Micro Frontend E-Commerce Application using React / Next.js where multiple independent frontend applications are interconnected and work together as a single scalable e-commerce platform.


Modules:

* User Authentication

* Product List

* Cart

* Order List


Use:
   * React.js /Next.js  (Latest Stable Version)

   *  Micro Frontend Architecture

   * Separate Angular applications:

      * shell (Host App)

      * auth-app

      * product-app

      * cart-app

      * Order-app


Routing Rules

         * Each app run on diffrent port like

         3000, 3001 , 30002 etc

         * Open Product page first

         * When click on Product "add to cart"

         * If not logged in → Redirect to Login

         * After login → Redirect to Products

         * Products -> Cart -> Orders (Protected Routes)



 Expected Technical Concepts

            * React.js (Hooks, Context, Lifecycle)
            * Next.js (App Router / Pages Router, SSR, SSG)
            * Redux Toolkit / Zu stand / Recoil
            * REST APIs / Axios / Fetch / Graph QL
            * Build Tools/ Webpack / Vite
            * JavaScript (ES6+) / TypeScript
            * Micro frontend Architecture/ Webpack Module Federation
